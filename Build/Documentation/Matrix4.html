 { x : 0.0, y : 0.0, width : 0.0, height : 0.0 } The viewport's corners as shown in Example 1.

 0.0 The near plane distance in window coordinates.

    y : 0.0,
    width : 1024.0,
    height : 768.0
}, 0.0, 1.0);
